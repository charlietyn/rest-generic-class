# Introducción

**Rest Generic Class** es un paquete para Laravel que provee clases base de modelo, servicio y controlador para construir endpoints REST con el mínimo de boilerplate. Se centra en:

- Filtrado basado en consultas con una sintaxis estructurada `oper`.
- Carga segura de relaciones mediante allowlist.
- Selección de campos y paginación (incluida paginación por cursor).
- Listado jerárquico opcional para modelos auto-referenciados.
- Utilidades opcionales de permisos para Spatie Laravel Permission.

Si eres nuevo/a en el paquete, empieza con el [Inicio rápido](../01-getting-started/02-quickstart.md).
