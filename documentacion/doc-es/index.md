# Documentación de Rest Generic Class

Bienvenido/a a la documentación del paquete **Rest Generic Class**. Esta librería proporciona clases base y utilidades para construir endpoints REST con filtrado dinámico, carga de relaciones y listados jerárquicos en aplicaciones Laravel.

**Empieza aquí:**
- [Introducción](00-overview/00-introduction.md)
- [Requisitos](01-getting-started/00-requirements.md)
- [Inicio rápido](01-getting-started/02-quickstart.md)

## Compatibilidad

| Componente | Versión compatible |
| --- | --- |
| PHP | ^8.0 |
| Laravel (componentes Illuminate) | ^12.0 |
